# Cross Interview #5

The goal of the technical interview is to check the quality of learning on topics: Performance optimizations, React Compiler, Tools and extensions for debugging, React Security, SSR.

## Interview questions list

### React Performance Optimizations
  1. What is React memoization and how does it improve performance?
  2. How does the useMemo hook help with performance optimization?
  3. What is React.memo and its class component analogue?
  4. How do you use React.lazy and Suspense for code splitting?
  5. How do you profile and measure React app performance?
  6. How do you use the useCallback hook to prevent unnecessary re-renders?
  7. How do you use the useMemo hook to prevent unnecessary re-renders?
  8. How does the useOptimistic hook work?
  9. How does the useDeferredValue hook work?
  10. How do useTransition and startTransition work?

### React Compiler
  11. What is the React Compiler and what problems does it aim to solve?
  12. How does the React Compiler handle JSX transformations?
  13. How do you configure the React Compiler for a custom project setup?
  14. What are the implications of using experimental compiler features in production?

### Tools and Extensions for Debugging
  15. What is the React Developer Tools extension and how do you use it?
  16. How can you inspect and debug component state and props in the browser?
  17. What are some useful browser extensions for debugging React applications?
  18. How can StrictMode be used for debugging?

### React Security
  19. What are common security risks in React applications?
  20. How do you prevent cross-site scripting (XSS) in React?
  21. What is the risk of using dangerouslySetInnerHTML and how can it be mitigated?
  22. What are the security implications of third-party React libraries?

### SSR (Next.js)
  23. What is server-side rendering (SSR) and how does it differ from client-side rendering?
  24. What are the benefits and drawbacks of SSR in React?
  25. How can you optimize performance and SEO with SSR in Next.js?
  26. What is Incremental Static Regeneration (ISR) in Next.js and how does it work?
  27. What is Static Site Generation and how could it be used?
  28. What React DOM server APIs do you know and how do they work?
  29. What directives are used in React?
  30. What are server components?
