# Cross Interview #1

The goal of the technical interview is to check the quality of learning on topics: React concepts, React components, JSX, Components styling, State in components.

## Interview questions list

### React concepts
  1. What is React, and what are its main advantages and disadvantages?
  2. What architectural approaches are commonly used in React applications?

### JSX
  3. What is JSX in React, and why is it used?
  4. What are the key rules and limitations of writing JSX?
  5. How can you use React without JSX, and how do you create a React app without it?
  6. What is the purpose of the key prop when rendering lists in JSX?
  7. What are JSX embedded expressions, and what constraints apply to their usage?
  8. How do you conditionally render elements in JSX?

### Class and Functional Components
  9. How do you create class and functional components in React?
  10. What are props and state, and how do they differ in usage and principles?
  11. How is state managed in class versus functional components?
  12. What is state lifting and prop drilling, and when are they needed?
  13. How do sibling components communicate in React?
  14. What is the difference between presentational (dumb) and container (smart) components?
  15. How do you handle default props and prop validation in functional and class components?
  16. What is component composition, and how would you decompose a large component into small ones?

### Hooks
  17. What are React hooks, and how have they changed state and feature management?
  18. What are the rules for using hooks in React?

### useState
  19. How does the useState hook work, and what are its main concepts?
  20. How do you update state based on the previous value using useState?
  21. What is lazy initialization of state in useState?
  22. What is state batching in React?

### React DOM
  23. How does the flushSync method work in React?
  24. What is the Fragment component, and when should it be used?
  25. What is StrictMode, and how is it applied in React applications?
  26. How does the Activity component work?
  27. What is the difference between createRoot and hydrateRoot?
  28. What are portals in React, and when would you use them?

### Component Styling
  29. What approaches can be used to style React components?
  30. How can dynamic styles be created and applied using JavaScript?
