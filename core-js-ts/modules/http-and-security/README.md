# HTTP and Security

## In this module you will learn:

### Web Communication Protocols:
- HTTP
- HTTPS
- HTTP/1 vs HTTP/2 vs HTTP/3
- REST

### Common web-security knowledge:
- CORS
- OWASP Top 10
- XSS
- CSRF/XSRF
- Possible protections

## Approximate time to complete this module:

- Theory: 3+ hours
