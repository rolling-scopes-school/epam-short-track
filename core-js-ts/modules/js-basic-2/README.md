# JS Basic 2/3

## In this module you will learn:

### Objects:
- create
- manage properties
- built-in methods
- hash data structure
- property descriptors
- static methods
- getter/setter

### Closures:
- nested functions
- context
- inner/outer context

### Map, WeakMap

### Functions:
- create
- invoke
- arrow functions
- rest operator
- default parameters
- arguments
- ```this``` scope
- ```call```, ```apply``` methods
- recursion

## Approximate time to complete this module:
- Theory: 3 hours
- Additional materials: 10 hours
