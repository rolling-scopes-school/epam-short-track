# TS Basic

## In this module you will learn:

### Primitive Types:
- common primitive types
- type annotation
- type inference
- enums
- structural typing

### Basic type manipulation:
- union
- intersection
- composition
- Partial
- Required
- Readonly
- keyof
- typeof
- conditional types
- template literal type
- infer
- mapped types

### Arrays, Tuples, Objects, Records:
- named tuple
- multi type array
- object vs record
- as const

### Functions:
- overload
- generics
- call/construct signature
- this type

### Interfaces, Types, Generics:
- generic constraints
- index signature
- interface vs type

### Classes:
- abstract classes
- override
- access modificators
- inheritance

## Approximate time to complete this module:
- Theory: 3 hours
- Additional materials: 10 hours
