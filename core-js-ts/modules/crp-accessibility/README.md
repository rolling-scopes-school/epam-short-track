# Critical rendering path / Accessibility

## In this module you will learn:

### CRP:
- DOM
- CSSOM
- Render Tree
- Layout
- Paint
- Composite

### Accessibility:
- A11y
- A, AA, AAA
- HTML (semantic and tags)
- Keyboard navigation
- NVDA / VoiceOver
- Lighthouse / Axe



## Approximate time to complete this module:
- Theory: 3 hours
- Additional materials: 10 hours
