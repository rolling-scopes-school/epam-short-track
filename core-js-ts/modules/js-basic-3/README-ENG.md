### Objects:
- Creation

- Loops (for in, for of loops over keys and values)

- Property descriptors

- Classes
- Prototype
- Constructor
- Inheritance
- set/get
- Public, Static, Private methods
- Prototype inheritance.
- Super keyword
- Static keyword
- inheritance
- abstract classes
- generic classes
- access modificators
- Singleton pattern
- instanceof operator
