# Redux & NgRx

## In this module, students need to learn:

- `NgRx` library.
- Store state management: `Store`, `Actions`, `Reducers`, `Selectors`.
- Application with `Effects`.
- Advantages of `NgRx`.

## Approximate time to complete this module:

- Theory: 6 hours;
