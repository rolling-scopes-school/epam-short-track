# Routing

## In this module, students need to learn:

- Basics of routing.
- RouterOutlet directive.
- Multiple routing. Routing with parameters.
- `Router guards`.
- Asynchronous loading.

## Approximate time to complete this module:

- Theory: 1.5 hours
- Assignment: 1 hour
